# Verified Income Desk

A public-interest income literacy platform for tested income models, realistic
digital work pathways, local business service experiments, anti-scam education,
proof signals, and field-tested earning playbooks.

## Stack
- Next.js 14 (App Router) · TypeScript · Tailwind CSS
- Self-hosted fonts via @fontsource (Newsreader, IBM Plex Sans, IBM Plex Mono)

## Run
```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # production build (verified passing)
npm run start
```

## Routes
- `/` — homepage (hero + 9 sections)
- `/income-models/local-business-website-services` — featured model (IM-01)

## Design system
- Palette: graphite ink base, verdigris "verified" green, ochre caution, brick flag
- Signature device: the proof-signal status chip (Verified / Under test / Caution / Flagged)
- All content and copy live in `lib/site.ts`

## Notes
This is income literacy, not financial advice. No content promises earnings.
No health, hospital, patient, clinician, medical, symptoms, prevention,
specialist-centre, or Nigerian-health-context content is present.
